# lending-club
Predictive modeling project involving pre and post IPO lending club approved loans.

This project showcases various supervised machine learning techniques used to predict the grade of approved Lending Club loans from both the pre and post IPO period.

Data was collected for this project from the following source:
https://www.lendingclub.com/info/download-data.action

The current version of this project uses following quarterly data from the following time periods:
- 2014: Q1, Q2, Q3
- 2016: Q3, Q4
- 2017: Q1
